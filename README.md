# bcrm
Open Source Business CRM Solution

# developement

Create a virtual environment

```bash
mkvirtualenv bcrm
```

Install the dependencies

```
cd src
pip install -r requirements.txt
```

Set the flask app root

```bash
FLASK_APP=app
```

Run the application

```bash
flask run
```

open the following url http://127.0.0.1:5000
